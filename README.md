## ![Cabbage](assets/banner.png)

A local-first, Git-powered Markdown notes application for personal knowledge management. It uses a standard Git repository as the single source of truth.

## Core philosophy

* Git as storage
* Local-first design
* No cloud and no server
* Plain Markdown files only

## Features

* Markdown notes
* Git-based synchronization
* Offline-first
* Cross-platform desktop application

## Tech stack

* Tauri
* Rust
* Svelte
* CodeMirror 6
* Git

## Usage concept

* Notes are standard Markdown files (.md).
* A folder (vault) is a standard Git repository.
* Synchronization is handled via standard Git operations (push/pull).
* There are no user accounts or centralized servers.

## Disclaimer

* No cloud integration
* No user accounts
* No backend server
* Everything is local and strictly Git-based
