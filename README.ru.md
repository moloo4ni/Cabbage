## ![Cabbage](assets/banner.png)

Локальное приложение для ведения заметок в формате Markdown, использующее Git в качестве системы хранения. Стандартный Git-репозиторий выступает единственным источником истины для вашей базы знаний.

## Core philosophy

* Git как хранилище данных
* Local-first архитектура
* Отсутствие облака и сервера
* Только обычные Markdown файлы

## Features

* Заметки в формате Markdown
* Синхронизация на базе Git
* Offline-first подход
* Кроссплатформенное desktop-приложение

## Tech stack

* Tauri
* Rust
* Svelte
* CodeMirror 6
* Git

## Usage concept

* Заметки — это стандартные файлы Markdown (.md).
* Папка (vault) — это стандартный Git-репозиторий.
* Синхронизация выполняется через базовые операции Git (push/pull).
* Никаких пользовательских аккаунтов или централизованных серверов.

## Disclaimer

* Никакой облачной интеграции
* Никаких аккаунтов
* Никакого бэкенд-сервера
* Все данные хранятся локально и управляются исключительно через Git
